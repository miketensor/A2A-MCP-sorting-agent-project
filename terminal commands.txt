Project Setup
---------------------------
# 1. Create project folder
mkdir file-sorter-agent
cd file-sorter-agent

# 2. Create virtual environment
python -m venv .venv

# 3. Activate it
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate

# 4. Install dependencies
pip install groq fastapi uvicorn httpx python-dotenv mcp
pip install -r requirements.txt
pip freeze > requirements.txt

# 5. Open in VS Code
code .

> VS Code Configuration
.vscode/settings.json
.vscode/launch.json

> .env

Github From root folder
------------------------
git init
> Create a .gitignore file
> Stage and commit your code
git add .
git commit -m "Initial commit"
> Create repo and push
gh repo create A2A-MCP-sorting-agent-project --public --source=. --remote=origin --push


Execute
------------------------
python sorting_agent.py
python agent.py ./test_dir


Commit changes
------------------------
git status
> stage everything
git add .
> commit 
git commit -m "Update: describe your changes"
> branch main
git checkout main 
> push to Git
git push origin main

New branch
------------------------
git checkout main
> load latest version
git pull origin main
> create a new branch
git checkout -b MCP-with-A2A
> Push to github
git push -u origin MCP-with-A2A

Pull Request
-----------------------
> create PR
gh pr create --base main --head MCP-with-A2A --title "Add MCP with A2A integration" --body "This PR adds the MCP agent integration with A2A. Includes initial unit tests and requirements updates."
> merge
gh pr merge MCP-with-A2A --merge
> update locally
git checkout main
git pull origin main
